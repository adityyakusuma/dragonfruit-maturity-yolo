# Dragon Fruit Maturity Detection (YOLO)

Website deteksi kematangan buah naga menggunakan algoritma YOLO.

## Fitur
- Upload gambar buah naga
- Deteksi tingkat kematangan
- Tampilan web berbasis Flask

## Tech Stack
- Python
- Flask
- YOLO
- OpenCV

## Cara Menjalankan (Local)
```bash
pip install -r requirements.txt
python app.py
